# mango-kit
mango mei的工具集

详细文档见 - http://kitdoc.meiflower.top            

## mango-validator-kit 
* 集成fluent-validator,hibernate-validator，并实现特定业务场景的后台验证框架工具。

## mango-time-kit 
* 实现Java方法运行耗时监控工具。

## mango-excel-kit
* 1.基于POI，实现java excel导出文档工具。
* 2.基于xls模板，填写动态数据思想。（单行数据填入和方格数据填入）
